## ho to use function 

this is the code and how to use this 

```go
package linkedlist

func testing(){
	a:=createLinkedList("A")
	a.insertRight("B")
	a.insertRight("C")
	a.insertLeft("D")
	a.showLinkedList()
}
```

output

```bash
D->A->B->C
```
